API Endpoints

Get All Venues

Returns a list of all available venues.

GET /api/v1/venues

Get Venue Details

Returns details for a specific venue.

GET /api/v1/venues/{venueId}

Get Venue Events

Returns events for a specific venue.

GET /api/v1/venues/{venueId}/events

Get All Events

Returns events from all venues.

GET /api/v1/venues/events/all
